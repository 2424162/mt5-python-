mt5品种窗体截屏接口

描述

在编写的EA文件中需要调用 写文件0708.mqh ,调用Weiyou类中函数——截屏()

函数

string 截屏() //return 图片文件名

注释

不需要填任何参数，EA调用本函数后返回加载EA的品种窗体的截图文件名,如果需要通过邮件微信发送截到的图片只需要将图片文件名变量填写在相应位置

例如

#property copyright "Copyright 2019, MetaQuotes Software Corp."

#property link "https://www.mql5.com"

#property version "1.00"

#include<写文件0708.mqh> //导入mqh文件

input string friend; //定义接受人

input string youxiang;//定义接收邮箱

int OnInit()

{

      Weiyou xiewenjian;//类对象

      string 图片= xiewenjian.截图(); //调用截图函数将图片名称赋值给变量

      xiewenjian.微信("",friend,图片); //发微信图片

      xiewenjian.邮箱("",friend,图片);//发邮件图片

      return(INIT_SUCCEEDED);

}
